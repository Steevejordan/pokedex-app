 Fonctionnalités mises en place :
Connexion via un token d’authentification.

Affichage des Pokémon découverts.

Page de détails pour chaque Pokémon.

Capture individuelle ou automatique des Pokémon découverts.

Upload d’une image pour découvrir de nouvelles espèces (via l’API).

 Remarque importante :
Certaines fonctionnalités ne sont visibles que si des Pokémon ont été découverts avec le jeton utilisé. Par exemple, les images et les détails ne s’affichent pas si l’espèce n’a pas encore été identifiée via l’API.
